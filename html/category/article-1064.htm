代码编写

打开我们创建的1.py文件

from PIL import Image import sys   for arg in sys.argv:     a = sys.argv[1] # 获取图片文件的绝对路径  img = Image.open(a) resize = 150,150 img_resize = img.resize(resize) # 图片转换 img_resize.save("C:/Users/Adminisrator/Desktop/1/after/a.png")# 图片保存

代码写好之后，保存
此时不要运行，因为会报错
这时将图片文件拖到代码文件上面，此时你就会发现图片会自动处理了

提醒

在将图片文件拖到代码文件上面时，先将Pillow模块安装一下，否则会报错。
